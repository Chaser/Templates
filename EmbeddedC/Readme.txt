/Project Directory (Application and Library)
_________________
 |	
 |
 |_____/bin 	(Output - Executable or library output)
 |
 |
 |_____/build 	(Contains object files, removed on a clean)
 |
 |
 |_____/docs	(Project Docs)
 |
 |
 |_____/src 	(All *.c and *.cpp project files. If the project is not a library then also include header files)
 |
 |
 |_____/include (Distributable include files.All necessary third-party header files that do not exist under /usr/local/include are also placed here)
 |	|
 |	| 
 |	|_____/project_name
 |
 |
 |___/lib 	(Compiled Library files)
 |
 |
 |___/tests 	(Test files)
